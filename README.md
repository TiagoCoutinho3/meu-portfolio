## 🛠️ Tecnologias & Ferramentas

- **Frontend:** React.js, TypeScript e Tailwind CSS (Design System customizado).
- **Backend:** Node.js para integração de dados e lógica de servidor.
- **Deploy:** Vercel (CI/CD integrado ao GitHub).
- **Arquitetura:** Componentização avançada e tipagem estrita com TS.
